# GIT it!

Scenario-driven Git learning platform. `frontend/` is a React + Vite SPA; `backend/` is Django + DRF (simulator, evaluator, persistence).

## Design Context

- [PRODUCT.md](PRODUCT.md) — register (product), users, brand personality ("arcane, immersive, premium"), anti-references, and design principles. Read before any UI/UX work.
- [DESIGN.md](DESIGN.md) — the visual system ("The Arcane Observatory"): tokens, colors, typography, elevation, component specs, and do's/don'ts. New screens must stay on this system.
